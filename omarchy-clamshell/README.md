# omarchy-clamshell

Small dotfiles project for Omarchy clamshell mode, dynamic workspace routing, and Waybar monitor-aware workspace display.

Project files live here in dotfiles:

- `.config/omarchy/clamshell.conf`
- `.config/omarchy/scripts/clamshell.sh`
- `.config/omarchy/scripts/clamshell-events.sh`
- `.config/omarchy/scripts/apply-waybar-layout.sh`
- `.config/omarchy/examples/*`

Runtime paths stay under `~/.config/omarchy/...`.

Detailed runtime documentation lives in:

- `.config/omarchy/README.md`
